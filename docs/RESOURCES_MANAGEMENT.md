# مدیریت منابع آموزشی

## مرور کلی

این سیستم شامل دو صفحه جداگانه برای مدیریت منابع آموزشی است:

1. صفحه مخصوص ادمین برای مدیریت منابع عمومی و اختصاصی مدارس
2. صفحه مخصوص معلمان برای ایجاد و مدیریت منابع مدرسه

## ویژگی‌ها

### برای ادمین:

- ایجاد منابع عمومی برای تمام مدارس
- ایجاد منابع اختصاصی برای یک مدرسه خاص
- فیلتر کردن منابع بر اساس مدرسه، پایه، درس
- جستجو در بین تمام منابع
- ویرایش و حذف هر منبعی در سیستم
- مشاهده اطلاعات فایل (نام، حجم، فرمت، تاریخ آپلود)

### برای معلمان:

- ایجاد منابع اختصاصی برای مدرسه خود
- فیلتر کردن منابع مدرسه بر اساس پایه و درس
- جستجو در بین منابع مدرسه و منابع عمومی
- ویرایش و حذف فقط منابعی که خودشان آپلود کرده‌اند
- تنظیم سطح نمایش منابع (قابل مشاهده برای دانش‌آموزان مدرسه / فقط معلمان / فقط خود معلم)

## ساختار داده

### جدول Resources

- `id`: شناسه یکتا
- `title`: عنوان منبع
- `file_url`: آدرس فایل
- `file_name`: نام فایل
- `file_size`: حجم فایل
- `file_type`: نوع فایل
- `grade_level`: پایه
- `subject`: درس
- `description`: توضیحات
- `school_id`: شناسه مدرسه (برای منابع مخصوص مدرسه)
- `uploaded_by`: شناسه کاربر آپلودکننده
- `visibility_level`: سطح نمایش (public, school, class, private)
- `created_at`: تاریخ ایجاد
- `updated_at`: تاریخ به‌روزرسانی

## API Endpoints

### برای معلمان:

- `GET /api/teacher/resources` - دریافت منابع مدرسه و منابع عمومی
- `POST /api/teacher/resources` - ایجاد منبع جدید
- `PUT /api/teacher/resources/:id` - به‌روزرسانی منبع (فقط منابع خود معلم)
- `DELETE /api/teacher/resources/:id` - حذف منبع (فقط منابع خود معلم)

### برای ادمین:

- `GET /api/admin/resources` - دریافت تمام منابع با قابلیت فیلتر
- `POST /api/admin/resources` - ایجاد منبع (عمومی یا مخصوص مدرسه)
- `PUT /api/admin/resources/:id` - به‌روزرسانی هر منبعی
- `DELETE /api/admin/resources/:id` - حذف هر منبعی

## صفحات UI

### صفحه معلمان (`/dashboard/teacher/resources`)

- فرم آپلود منبع با فیلدهای عنوان، پایه، درس، توضیحات
- فیلترهای جستجو بر اساس پایه و درس
- لیست کارتی منابع با امکان دانلود و حذف (برای منابع خود معلم)

### صفحه ادمین (`/admin/resources`)

- فرم آپلود منبع با امکان انتخاب مدرسه برای منابع مخصوص مدرسه
- فیلترهای پیشرفته شامل مدرسه، پایه، درس
- لیست کارتی منابع با امکان دانلود و حذف برای تمام منابع
